library(ecocomDP)

Converting an ecocomDP dataset to another model format can be as simple as calling a single function. Currently, we support conversion to the Darwin Core Archive (DwC-A) Event Core.

DwC-A Event Core

mypath <- paste0(tempdir(), "/dwca")
dir.create(mypath)

convert_to_dwca(
  path = mypath, 
  core_name = "event", 
  source_id = "edi.193.4", 
  derived_id = "edi.834.1", 
  user_id = "ecocomdp",
  user_domain = "EDI")
